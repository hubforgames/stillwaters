# 🌊 Still Waters

> *A breathing journey to calm the nervous system and expand your awareness.*

Still Waters is a mobile-first mindfulness game that guides you through breathing cycles, healing sound frequencies, and consciousness expansion — all in a single HTML file. No app store. No install. Just open and breathe.

---

## ✨ Features

- **4 Consciousness Levels** — Awakening · Flowing · Transcending · 🌑 Night Realm (secret unlock)
- **Guided Breathing Orb** — visual orb expands and contracts with your inhale / hold / exhale
- **Healing Frequencies** — ambient tones based on Solfeggio frequencies per level (174Hz → 285Hz → 396Hz → 528Hz)
- **Haptic Feedback** — gentle vibrations sync with each breath phase on mobile
- **Touch Ripples** — tap anywhere during a session to release tension as ripple waves
- **Session History** — tracks all past sessions with presence scores and breath counts
- **Zero Dependencies** — one single `index.html` file, works offline after first load

---

## 🎮 How to Play

1. Tap **Begin Journey**
2. Follow the breathing orb — expand with the inhale, pause on hold, release on exhale
3. Tap the screen during breathing to create ripple effects
4. Complete 7 breath cycles to ascend to the next level
5. Finish all 3 levels to unlock the secret **Night Realm**

---

## 🌑 Night Realm

Complete one full journey to unlock the secret 4th level. Features:
- Longer breath cycles (7s / 3.5s hold / 7s)
- Deep violet visuals
- 528Hz "transformation" frequency
- Void consciousness mantras
